# 로그인,로그아웃,회원가입 구현

## 로그인 과정
- 사용자가 폼에 id와 pwd를 입력
- 입력한 id,pwd에 대한 유효성검사
- 데이터베이스에서 해당 id와 pwd를 가진 user가 있는지 확인
- 세션에 유저에 대한 정보를 저장

## 로그아웃
- 세션에서 유저에 대한 정보를 제거

## 회원가입
- 폼으로 정보를 입력받고
- 유효성검사
- db에 저장
- 해당 유저 로그인(세션에 정보 저장)

## pj01 - FBV, no폼클래스

### 로그인
- 커스텀 유저모델 사용
- 유효성 검사 : 간단하게 글자수만 검사
- db에서 유저 확인
- 세션에 저장

### 로그아웃
- 세션에서 유저 제거

### 회원가입
- 유효성검사
- db에 저장
- 로그인

- 로그인 인증 : 기존 User모델 사용
